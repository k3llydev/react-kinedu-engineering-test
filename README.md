# Front-end Engineer test

## Instructions

- Clone the repository
- Execute `npm install`

Run application:

```js
npm run start:dev
```

Build:

```
npm run build:dev
```

> For production, simply change `:dev` with `:prod`

## Techincal documentation

Dependencies:
- React@16
- React DOM
- Webpack
- SASS
- Babel
- Jest
